# Které hodnoty zobrazí cyklus while?

Pro oba cykly si zapište hodnoty, které budou vypsány, a pak je porovnejte s řešením.

Vypíše `alert` v obou cyklech stejné hodnoty, nebo ne?

1. Prefixová forma `++i`:

    ```js
    let i = 0;
    while (++i < 5) alert( i );
    ```
2. Postfixová forma `i++`

    ```js
    let i = 0;
    while (i++ < 5) alert( i );
    ```
