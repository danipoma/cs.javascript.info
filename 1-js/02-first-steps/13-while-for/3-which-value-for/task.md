# Které hodnoty zobrazí cyklus „for“?

Pro oba cykly zapište hodnoty, které budou zobrazeny, a pak je porovnejte s řešením.

Vypíše `alert` v obou cyklech stejné hodnoty, nebo ne?

1. Postfixová forma:

    ```js
    for (let i = 0; i < 5; i++) alert( i );
    ```
2. Prefixová forma:

    ```js
    for (let i = 0; i < 5; ++i) alert( i );
    ```
